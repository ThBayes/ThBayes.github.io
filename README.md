# Simple Academic Website Template

This repository provides a streamlined template for building a professional academic website in under 20 minutes. 

## Getting Started

If you want to build your personal website using this template, a step-by-step guide is available in this [blog post](https://valentinkil.github.io/blog/website/).

## Credits and Motivation

None of the content in this repository or the associated blog post is entirely new; the project is heavily inspired by, or directly adapted from, the popular [al-folio Jekyll template](https://github.com/alshedivat/al-folio?tab=readme-ov-file). 

While the original al-folio template is excellent, it offers a multitude of options that are unnecessary for most academics and can make website creation overly complex. The value of this version lies in its simplicity: it provides a clean, professional result that is easy to maintain and meets the essential needs of most researchers without the overhead of more complex solutions.
